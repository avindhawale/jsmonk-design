Introduction

JSMonk UI is a premium component material with a fresh, new design inspired by Imaginary Design. It is based on the new css design paradigm and comes packed with multiple components. All components are built to fit perfectly with each other while aligning to the material concepts.

Github Access

Click here to access JSMonk UI repository.
Follow the development closely, access to the release history, hot bug fixes and new features. Drop a message before fork it or use it as a template to update easily.

No Dependencies

No other third party framework like NodeJS required to install. Just a reference and ready to use the components.

Responsive

Not yet fully responsive but can be used at some level. We will soon bring new version with fully responsive support.

Well Documented Code

Well documented and commented source to understand the code easily.

Browser Support

It supports all kind of browsers and all the versions. Except IE browser.

Free Update

Free lifetime updates including new features and bug fixes.

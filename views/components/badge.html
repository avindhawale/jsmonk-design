Badges

Badges can be used as part of links or buttons to provide a counter.

Avatar status badge

Avatar badges can be used to represent the use status and availability. You can use badge class to get badge on avatar and to show status use once the class busy or available or offline.

You can change the badge position with the help of badge-tl or badge-tr or badge-bl these classes as shown in below examples.

Themes

JSMonk UI provides a sensible default theme with a very generous set of values to get you started, but don't be afraid to change it or extend; you're encouraged to customize it as much as you need to fit the goals of your design.

Theme Configuration

Currently we are providing three different kinds of themes by default. They are default, dark and dracula themes. By default, these colors are inherited by the backgroundColor, textColor, and borderColor core plugins. Apart from these themes you can customize each component as per your requirements.

To use default provided themes just provide class name theme-default or theme-dark or theme-dracula to your root element in your html file.
